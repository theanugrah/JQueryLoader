    ABOUT :

  • This jQuery plugin has been developed to be used to represent ongoing process when the user clicks a link, also called a loading animation. The plugin supports a variety of customisation for excellent presentation.
    It does not use any image and hence is very light-weight.


  • Do check the demos by clicking on the menu on the left side before going through the features and how to use section.

    FEATURES :

  • Supports a variety of animation type : fade, zoom, wave, shake, shrink, hammer, bubbleFade, bubbleZoom.
  • Supports a range of customisation : bar size, color, shadow, speed, delay, background transparency, background color etc.
  • Very light wieght. Does not use any image.
  • Supports multiple loaders on the same page.
  • Easily attachable to variety of elements.
  • Automatically positions itself to the center of the elements attached to.
  • Very easy to use.
  • Free of cost.

    HOW TO USE :

  • The parent element to contain the loader must have css position attribute set.
  • Call the plugin on the 'id' of the element you want to attach it to.
    Ex: $('elementA#element_id').Loading();
    This attaches and runs the fade animation on top of the elementA.
  • To run a customised loader :
    Ex: $('elementA#element_id').Loading({parameter:value, parameter:value[,....]});
  • To stop the animation, call the $.Loading.stopLoader() function and pass it the element whose animation you want to stop.
    Ex: $.Loading.stopLoader($('elementA#element_id'));
    This stops the currently running animation on elementA.
    NOTE : Animation must already be attached prior to invoking $.Loading.stopLoader() function.

    CUSTOMISATION PARAMETERS :

  • Various parameters to customise the loader are :
    height : Any non negative value in pixels for bar height. width : Any non negative value in pixels for bar width.
    barColor : Any valid color format for color of bars. barShadow : html box-shadow equivalent parameter.
    radius : Radius for bars corners in pixels. spacing : Spacing between the bars.
    backgroundColor : The color of the background stage. backgroundOpacity : Transparency of the background.
    population : The total number of bars. delay : The delay of animating the successive bars.
    fadeLow : Lowest opacity during the animation. fadeHigh : Maximum opacity during the animation.
    newHeight : Maximum height during animation when using 'zoom', 'wave', 'bubbleZoom' and 'shrink' loader_type.
    newWidth : Maximum width when using 'zoom' loader_type.
    loader_type : Type of animation you want : fade, zoom, wave, shrink, shake, bubbleZoom, bubbleFade amd hammer.
    duration : The duration of a complete cycle of animation of a single bar in milliseconds.
  • These parameters can be adjusted independently to acheive beautiful animations.
    Ex: $('elementA#element_id').Loading({height:'5px', width:'5px', spacing:'1px', population:50, loader_type:'wave', duration:200});
    This will start wave loader with bar of size 5x5, spacing of 1px, total bars are 50 and duration of animation on each bar is 200ms.
